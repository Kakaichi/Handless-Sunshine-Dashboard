Handless Sunshine Dashboard v1.1.2

Pacote Windows pronto para uso (administrador necessario).

## Novidades

- Melhor gerenciamento de Moonlight Web (status, requisitos Funnel e operacoes em background mais estaveis)
- Notificacoes toast redesenhadas (overlay no canto inferior direito)
- Removidas notificacoes inline que empurravam o layout da interface

## Instalacao

1. Baixe **Handless-Sunshine-Dashboard-1.1.2-win64.zip**
2. Extraia em uma pasta (ex.: `C:\HandlessSunshineDashboard`)
3. Execute **HandlessSteam.exe** como administrador
4. Instale dependencias via `sunshine\instalar-dependencias.ps1 -InstallMissing` se for a primeira vez

Projeto vibecoded compartilhado publicamente — veja README no repositorio.
