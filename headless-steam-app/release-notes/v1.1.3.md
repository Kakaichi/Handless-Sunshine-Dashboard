Handless Sunshine Dashboard v1.1.3

Pacote Windows pronto para uso (administrador necessario).

## Novidades

- **Tela virtual (Experimental)** — jogo no monitor virtual (VDD) enquanto voce usa o PC no monitor fisico
- Configuracao guiada no app: instalar VDD, apontar Sunshine para o monitor virtual, sync de jogos
- Setup sem reiniciar o PC nem desligar o Sunshine manualmente
- **Acesso ao Desktop** — opcao em Jogos no Sunshine para mostrar ou ocultar Desktop no Moonlight (desativado por padrao)
- Correcoes de estabilidade do VDD (GUID do monitor virtual, numero DISPLAY subindo a cada ativacao)

## Instalacao

1. Baixe **Handless-Sunshine-Dashboard-1.1.3-win64.zip**
2. Extraia em uma pasta (ex.: `C:\HandlessSunshineDashboard`)
3. Execute **HandlessSteam.exe** como administrador
4. Na aba Sunshine, marque **Iniciar jogo em uma tela virtual** e siga os requisitos na tela

Projeto vibecoded compartilhado publicamente — veja README no repositorio.
